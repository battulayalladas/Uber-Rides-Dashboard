Power BI Dashboard analyzing Uber rides data
# 🚗 Uber Rides Dashboard | Power BI

## Business Problem
Understanding ride booking patterns, cancellations,
and peak usage times to improve service quality.

## Tool Used
- Power BI

## Visuals Created
- KPI Cards: Total Bookings, Customer Count
- Column Chart: Bookings by Time of Day
- Bar Chart: Cancelled Rides by Customer
- Donut Chart: Booking Status Distribution
- Category breakdown: UberXL vs other types

## Key Insights
- Identified peak booking hours
- Analyzed cancellation reasons and patterns
- Ride type preference by customers
- Booking status success vs failure rate
